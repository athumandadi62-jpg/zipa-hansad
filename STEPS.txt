Step-by-Step Deployment on Localhost

Step 1: Clone or Copy the Laravel Project
	git clone https://github.com/your-username/your-laravel-project.git
cd your-laravel-project


Step 2: Install Dependencies
	composer install
If using Laravel Mix (for assets):
	npm install
	npm run dev

Step 3: Set Up Environment File
	Copy .env.example to .env:
	cp .env.example .env
	Then edit .env to match your local setup:

APP_NAME=Laravel
APP_ENV=local
APP_KEY=
APP_DEBUG=true
APP_URL=http://localhost:8000

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=your_db_name
DB_USERNAME=root
DB_PASSWORD=your_password

Step 4: Generate Application Key
	php artisan key:generate


Step 5: Set Up Database

a.Create a new database in phpMyAdmin or via terminal.
b.Match its name in your .env file.
Then run:
	php artisan migrate:fresh --seed

Step 6: Run Laravel Server

You can use the built-in Laravel server:
	php artisan serve
This will serve your app at:
http://127.0.0.1:8000
